# 概要
オブジェクト指向によるプログラミングで制作したゲームです。  
コマンド選択式のRPGの戦闘のようなシステムです。  
このゲームには以下のような特徴があります。
・主人公選択  
・猫爪とビームによる攻撃  
・HP回復アイテムの使用  
・獲得したポイントにより称号が得られる  

# 開発環境
HTML, CSS, PHP, JavaScript(JQuey)
